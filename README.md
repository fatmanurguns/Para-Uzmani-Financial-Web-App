# Para-Uzmani-Financial-Web-App
A personal finance tracking web application developed using HTML, CSS, and JavaScript.


Para Uzmanı - Personal Finance Web App
"Para Uzmanı" is a user-friendly web application designed to help users track their income, expenses, and overall budget management.

Key Features
Interactive UI: Clean and responsive design for a better user experience.

Expense Tracking: Add, edit, and categorize daily financial activities.

Dynamic Calculations: Real-time updates on total balance and savings using JavaScript.

Technologies Used
HTML5 & CSS3: For structure and styling.

JavaScript (Vanilla): For application logic and interactivity.

Developed as part of the Web Development curriculum at Başkent University
